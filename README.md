# Personal Portfolio Website

## Description
This is a responsive personal portfolio website built to showcase my projects, skills, and contact information.

## Features
- Responsive design for mobile & desktop
- About me, skills, and project showcase sections
- Contact form and social media links
- Clean and modern UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open `index.html` in any browser
